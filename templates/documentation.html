Documentación del proyecto

API REST con documentación OpenAPI automática disponible en /docs y /redoc.

Endpoints principales

  • /clients/html - GUI para listar clientes
  • /clients/create - Formulario para crear cliente (subir imagen opcional)
  • /instruments/html - GUI para listar instrumentos
  • /instruments/create - Formulario para crear instrumento (subir imagen opcional)
  • /orders/html - GUI para listar ventas
  • /orders/create - Formulario para crear venta
  • /dashboard - Dashboard con ventas por instrumento y por día

Soporte de imágenes

Las imágenes se guardan en /static/images/clients/ y /static/images/instruments/ y se acceden desde las páginas HTML mediante la ruta que empieza con /static/.
